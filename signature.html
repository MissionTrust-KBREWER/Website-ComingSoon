How to use in Outlook:
1. Click once inside the white signature area below.
2. Press Ctrl+A to select all, then Ctrl+C to copy.
3. Outlook → Settings → Mail → Compose and reply → New signature → Ctrl+V to paste.
4. Save and set as default.
